## Example Mod: "Frostlands Expansion"
## Demonstrates the full Veilborn modding API.
## Drop this folder in: user://mods/frostlands_expansion/
##
## Files in this mod:
##   manifest.json     ← Mod metadata and load order
##   biomes.json       ← New biome: "deep_glacier"
##   items.json        ← New items: frostbite weapons, glacier materials
##   creatures.json    ← New creature: "ancient_frost_wyrm"
##   recipes.json      ← New crafting recipes for frost gear
##   hooks.gd          ← GDScript event hooks
